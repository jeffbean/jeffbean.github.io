# scss theme


This was a combination of the default elements plus other elements from pygments elements


```css
/**
 * Syntax highlighting styles
 */
$string-color: #75715e;
$subheading-color: #f92672;
$name-color: #f8f8f2;
$keyword-color: #66d9ef;
$literal-string-color: #e6db74;
$number-number: #ae81ff;

.highlight {
    background: #333;
    @extend %vertical-rhythm;

    .highlighter-rouge & {
      background: #eef;
    }
    pre {
        background-color: #333;
        color: #f5f5f5
    }
    .c     { color: $string-color; font-style: italic } // Comment
    .err   { color: #960050; background-color: #1e0010 } // Error
    .k     { font-weight: bold; color: $keyword-color } // Keyword
    .o     { font-weight: bold; color: $subheading-color } // Operator
    .l     { color: $number-number } // Literal
    .cm    { color: $string-color; font-style: italic } // Comment.Multiline
    .cp    { color: $string-color; font-weight: bold } // Comment.Preproc
    .c1    { color: $string-color; font-style: italic } // Comment.Single
    .cs    { color: $string-color; font-weight: bold; font-style: italic } // Comment.Special
    .gd    { color: $subheading-color; background-color: #fdd } // Generic.Deleted
    .gd .x { color: $subheading-color; background-color: #faa } // Generic.Deleted.Specific
    .ge    { font-style: italic } // Generic.Emph
    .gr    { color: #a00 } // Generic.Error
    .gh    { color: #75715e } // Generic.Heading
    .gi    { color: #a6e22e; background-color: #dfd } // Generic.Inserted
    .gi .x { color: #a6e22e; background-color: #afa } // Generic.Inserted.Specific
    .go    { color: #888 } // Generic.Output
    .gp    { color: #555 } // Generic.Prompt
    .gs    { font-weight: bold } // Generic.Strong
    .gu    { color: $string-color } // Generic.Subheading
    .gt    { color: #a00 } // Generic.Traceback
    .kc    { font-weight: bold; color: $keyword-color } // Keyword.Constant
    .kd    { font-weight: bold; color: $keyword-color } // Keyword.Declaration
    .kp    { font-weight: bold; color: $keyword-color } // Keyword.Pseudo
    .kn    { color: $subheading-color } // Keyword.Declaration
    .kr    { font-weight: bold; color: $keyword-color } // Keyword.Reserved
    .kt    { color: $keyword-color; font-weight: bold } // Keyword.Type
    .ld    { color: $literal-string-color } // Literal.Date
    .m     { color: $number-number } // Literal.Number
    .s     { color: $literal-string-color } // Literal.String
    .n     { color: $name-color } // Name
    .na    { color: #a6e22e } // Name.Attribute
    .nb    { color: $name-color } // Name.Builtin
    .nc    { color: #a6e22e; font-weight: bold } // Name.Class
    .nd    { color: #a6e22e } // Name.Decorator
    .no    { color: $keyword-color } // Name.Constant
    .ni    { color: $name-color } // Name.Entity
    .ne    { color: #a6e22e; font-weight: bold } // Name.Exception
    .nl    { color: $name-color } // Name.Label
    .nf    { color: #a6e22e; font-weight: bold } // Name.Function
    .nn    { color: $name-color } // Name.Namespace
    .nt    { color: $subheading-color } // Name.Tag
    .nv    { color: $name-color } // Name.Variable
    .nx    { color: #a6e22e } /* Name.Other */
    .py    { color: $name-color }/* Name.Property */
    .ow    { color: $subheading-color; font-weight: bold } // Operator.Word
    .p     { color: $name-color }
    .w     { color: $name-color } // Text.Whitespace
    .mf    { color: $number-number } // Literal.Number.Float
    .mh    { color: $number-number } // Literal.Number.Hex
    .mi    { color: $number-number } // Literal.Number.Integer
    .mo    { color: $number-number } // Literal.Number.Oct
    .sb    { color: $literal-string-color } // Literal.String.Backtick
    .sc    { color: $literal-string-color } // Literal.String.Char
    .sd    { color: $literal-string-color } // Literal.String.Doc
    .s2    { color: $literal-string-color } // Literal.String.Double
    .se    { color: $number-number } // Literal.String.Escape
    .sh    { color: $literal-string-color } // Literal.String.Heredoc
    .si    { color: $literal-string-color } // Literal.String.Interpol
    .sx    { color: $literal-string-color } // Literal.String.Other
    .sr    { color: $literal-string-color } // Literal.String.Regex
    .s1    { color: $literal-string-color } // Literal.String.Single
    .ss    { color: $literal-string-color } // Literal.String.Symbol
    .bp    { color: $name-color } // Name.Builtin.Pseudo
    .vc    { color: $name-color } // Name.Variable.Class
    .vg    { color: $name-color } // Name.Variable.Global
    .vi    { color: $name-color } // Name.Variable.Instance
    .il    { color: $number-color } // Literal.Number.Integer.Long
}
```
